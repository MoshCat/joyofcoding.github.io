Enjoying client-side development with AngularJS

There are a lot of Javascript frameworks out there, some good, some bad. There are so many of them, that it is probably safe to say that most <word>.js combinations are already taken :)

How to pick one of them for your (or your clients') next application? Most of them can build the TodoMVC application in no-time. But how does it hold up when implementing a real application? You can only find out by using the framework in ways that go beyond the regular tutorials.

In this 2 hour hands-on session you are going to get started with application development in AngularJS.
