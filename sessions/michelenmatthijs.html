An Introduction to Actors and Concurrency

Time: 14:30 - 16:20

Room: Branderij

Abstract

Writing concurrent software is traditionally considered very troublesome. Given only threads and locks this view is wholly justified. This workshop introduces the actor model: an alternative concurrency model (used by Scala and Erlang) which greatly simplifies concurrent programming.
